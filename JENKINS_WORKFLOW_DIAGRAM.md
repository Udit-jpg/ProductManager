## How to Use

1. **Push code to GitHub**
   ```bash
   git add .
   git commit -m "Your changes"
   git push origin main
   ```

2. **Jenkins automatically:**
   - Detects the push via webhook
   - Starts the pipeline
   - Builds all services
   - Runs tests
   - Generates reports
   - Sends email

3. **View reports:**
   - HTML Report: http://localhost:9090/job/ProductManager-Pipeline/[build-number]/Build_Report/
   - Test Results: http://localhost:9090/job/ProductManager-Pipeline/[build-number]/testReport/
   - Console Output: http://localhost:9090/job/ProductManager-Pipeline/[build-number]/console

4. **Check email:**
   - Success/failure notification sent automatically
   - Includes build summary and links
